REPOSITORIO es un sistema de almacenamiento digital centralizado diseñado para almacenar, organizar, preservar y compartir información, archivos o código fuente. Garantiza el acceso y la gestión de datos, desde proyectos de software hasta documentos académicos y empresariales.COMMIT es un punto de control, o una "fotografía", del estado de tus archivos en un momento específico. Es un término fundamental en los sistemas de control de versiones como Git (usado en plataformas como GitHub o GitLab) y en bases de datos. PUSH es el comando utilizado para enviar los cambios confirmados (commits) de tu repositorio local a un repositorio remoto (como GitHub, GitLab o Bitbucket). Es el paso final para respaldar tu código en la nube o compartirlo con tu equipo.
